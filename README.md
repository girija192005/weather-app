# 🌤️ Weather App

A full stack weather app built with React + Node.js + OpenWeather API.

## 🚀 Features
- 🔍 Search any city worldwide
- 🌡️ Current temp, humidity, wind speed
- 📅 5-day weather forecast
- 🔐 API key secured in backend

## 🛠️ Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- API: OpenWeatherMap

## ⚙️ Setup
1. Clone the repo
2. Add your API key in `backend/.env`
3. Run `node server.js` in backend
4. Run `npm run dev` in frontend
